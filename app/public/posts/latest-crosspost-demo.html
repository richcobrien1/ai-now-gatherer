How to view this demo locally

Run a static server from the repo root and open the file in your browser. Examples:

# Using Node http-server (recommended)
npx http-server app/public -p 8080

# Or using Python (if available)
python -m http.server 8080 --directory app/public

# Then open in your browser:
# http://localhost:8080/posts/latest-crosspost-demo.html

One-click POST to n8n (manual)

If your n8n webhook is reachable, copy the file latest-crosspost.json and POST to /webhook/youtube-latest. Example curl:

curl -X POST "https://your-n8n.example.com/webhook/youtube-latest" \
  -H 'Content-Type: application/json' \
  --data-binary @latest-crosspost.json
